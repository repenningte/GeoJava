# GeoJava
Teresa e Raphael Plump

Projeto desenvolvido em Java com o objetivo de aplicar conceitos de orientação a objetos, separação de responsabilidades entre classes e cálculos de geometria plana e espacial.

Tecnologia utilizadas:
○ Java
○ VS Code

Funcionalidades:
 - Cálculo de áreas
 - Cálculo de volumes
 - Organização em classes específicas para geometria plana e espacial
 - Execução por classe principal (app.java)
